## Pitfalls Discovered During Development

1. **`false` vs `False` in TOOLS dicts**: Python TOOLS arrays are dicts, not JSON. `"default": false` → `NameError`. Use `False`.
2. **`replace_all=true` corruption**: Replacing a string that appears in both a tool func and HTTP handler breaks indentation. Use unique context.
3. **Windows `sed -i` literal `\n`**: MSYS sed doesn't interpret `\n` in insert mode. Use Python for multi-line edits.
4. **Plugin registration ≠ tool availability**: After patching `__init__.py`, `/reset` doesn't reload plugin code. Need **full Hermes restart**.
5. **`_load_state`/`_save_state` symmetry**: Every new global state variable needs entries in both functions.
